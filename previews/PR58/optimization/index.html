Parameter Optimization

Selectively Optimizing Parameters

In the Basis Sets section, we have briefly introduced the parameters in terms of ParamBox that are embedded in containers such as BasisFunc and BasisFuncs directly used to form a basis set. This means how we construct the basis set using the parameters will determine the parameter space for the basis set optimization. For more information please refer to Constructing basis sets based on ParamBox.

Here is an example of using GaussFunc and GridBox to quickly generate a grid-based basis set with only 3 independent parameters. One is the spacing $L$ of the grid points that indirectly determines all the center coordinates of the basis function through a series of mapping functions; the other two are one exponent coefficient $\alpha$ and one contraction coefficient $d$.


julia> nuc = ["H", "H"]2-element Vector{String}:
 "H"
 "H"
julia> nucCoords = [[-0.7,0.0,0.0], [0.7,0.0,0.0]]2-element Vector{Vector{Float64}}:
 [-0.7, 0.0, 0.0]
 [0.7, 0.0, 0.0]
julia> grid = GridBox(1, 3.0)GridBox{8}(num, len, coord)
julia> gf1 = GaussFunc(0.7,1)GaussFunc(xpn=ParamBox{Float64, :α, Quiqbox.FLevel{1, 0}}(0.7)[∂][α], con=ParamBox{Float64, :d, Quiqbox.FLevel{1, 0}}(1.0)[∂][d])
julia> bs = genBasisFunc.(grid.box, Ref([gf1]))(BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][-1.5, -1.5, -1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][-1.5, -1.5, 1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][-1.5, 1.5, -1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][-1.5, 1.5, 1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][1.5, -1.5, -1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][1.5, -1.5, 1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][1.5, 1.5, -1.5], BasisFunc{0, 1}(center, gauss)[X⁰Y⁰Z⁰][1.5, 1.5, 1.5])

After constructing the basis set, we need to use markParams! to mark all the unique parameters that can also be optimized later:

julia> pars = markParams!(bs, true)3-element Vector{ParamBox{Float64, V, FL} where {V, FL<:Quiqbox.FLevel}}:
 ParamBox{Float64, :X, Quiqbox.FLevel{2, 0}}(3.0)[∂][L₁]
 ParamBox{Float64, :α, Quiqbox.FLevel{1, 0}}(0.7)[∂][α₁]
 ParamBox{Float64, :d, Quiqbox.FLevel{1, 0}}(1.0)[∂][d₁]

As expected, there are only 3 unique tunable independent parameters despite the basis set having 8 basis functions with a total of 40 parameters. However, if we take a step further, we can remove $d$ since each basis function here is just one same Gaussian function. Thus, input the left parameters (along with other necessary arguments) into the optimizeParams! and we can sit and wait for the optimization iterations to complete.

Store Customized Basis Set

You can also store the information of the basis set in GTBasis which contains not only the basis set but also the related 1-electron and 2-electron integral values. GTBasis can also be an input argument for runHF to save the time of recalculating the integrals of basis sets.

julia> GTBasis(bs)GTBasis{8, BasisFunc{0, 1}, Float64}(basis, S, Te, eeI, getVne, getHcore)
